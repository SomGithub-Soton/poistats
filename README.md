## 🎯 Overview

**poistats** is a lightweight R package for Poisson distribution analysis. It provides simple, intuitive functions to:

- ✨ Generate random Poisson-distributed numbers
- 📈 Calculate comprehensive summary statistics
- 📊 Create beautiful visualizations (histogram, boxplot, violin plot, density plot)

Perfect for teaching, research, and quick statistical analysis!
